# people-api

## How to run application
In a terminal run (Linux, Unix or Mac)
Tests
```
./mvnw test
```

Run api
```
./mvnw clean spring-boot:run
```

In a Command window run (Windows)
Tests
```
mvnw.cmd test
```

Run api
```
mvnw.cmd clean spring-boot:run
```
