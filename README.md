# people-api

## How to run application
In a terminal on Linux, Unix or Mac.
Tests.
```
./mvnw test
```

Run api.
```
./mvnw clean spring-boot:run
```

In a Command window on Windows Operating Systems.
Tests
```
mvnw.cmd test
```

Run api
```
mvnw.cmd clean spring-boot:run
```
